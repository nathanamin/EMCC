How to clone a respository:
1. git clone https://github.com/nathanamin/EMCC.git
2. git remote -v
(If you don’t have an origin, then follow the next step)
3. git remote add origin https://github.com/nathanamin/EMCC.git

How to Push:
1. git add .
2. git commit -m “First Commit”
3. git remote add origin https://github.com/nathanamin/EMCC.git
4. git remote -v
5. git push -u origin master
